# Diabetes Prediction Model

## Overview
This project builds machine learning models to predict diabetes using health-related features and evaluates performance across multiple algorithms.

## Dataset
- CDC diabetes dataset
- Includes health indicators such as BMI, activity, and medical history

## Models Used
- Logistic Regression
- SVM
- Decision Tree
- Random Forest
- AdaBoost

## Key Features
- Data preprocessing and scaling
- Train/test split with stratification
- ROC curve and AUC evaluation
- Feature importance analysis
- Model comparison across metrics

## Tech Stack
- Python
- scikit-learn
- pandas, numpy
- matplotlib

## Key Takeaways
- Built classification models on real-world healthcare data
- Evaluated model performance using multiple metrics
- Identified key predictors influencing outcomes
